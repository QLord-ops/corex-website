## 🚀 Как использовать

### Переключение языков
1. **Десктоп:** Нажмите на кнопку с кодом языка (EN/DE) в Header
2. **Мобильная версия:** Откройте меню и выберите язык внизу

### Добавление новых переводов
1. Откройте `frontend/src/i18n/translations.js`
2. Добавьте ключ в объекты `en` и `de`
3. Используйте в компонентах через `t('your.key')`

Пример:
```javascript
// В translations.js
en: {
  newSection: {
    title: "New Title"
  }
}

// В компоненте
const { t } = useTranslation();
<h1>{t('newSection.title')}</h1>
```

## ⚙️ Настройка Google Analytics 4

1. Перейдите на https://analytics.google.com
2. Создайте новое свойство GA4
3. Получите Measurement ID (формат: `G-XXXXXXXXXX`)
4. Откройте `frontend/public/index.html`
5. Найдите строку: `gtag('config', 'G-XXXXXXXXXX', {`
6. Замените `G-XXXXXXXXXX` на ваш реальный ID

## 📊 Структура переводов

Все переводы организованы по секциям:
- `header.*` - навигация
- `entry.*` - главный экран
- `pain.*` - болевые точки
- `how.*` - как это работает
- `proof.*` - доказательства
- `decision.*` - финальное утверждение
- `action.*` - форма контакта
- `faq.*` - часто задаваемые вопросы

## 🔄 Обновление HTML lang атрибута

HTML lang атрибут автоматически обновляется при смене языка через:
- `useTranslation` хук обновляет `document.documentElement.lang`
- Начальное значение устанавливается в `index.js`
